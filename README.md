# DELIVER ASSETS App Suite

Repositorio principal de las cuatro aplicaciones independientes de DELIVER ASSETS.

## Estructura prevista

- `apps/customer/` — aplicación del cliente
- `apps/business/` — aplicación para negocios
- `apps/rider/` — aplicación del repartidor
- `apps/control/` — aplicación de administración y control

Cada aplicación mantiene su propio identificador, dependencias, scripts, versión y proceso de compilación.

## Estado de carga

- Customer: en proceso de incorporación
- Business: pendiente
- Rider: pendiente
- Control: pendiente

No deben subirse `node_modules`, compilaciones Android/iOS, cachés, archivos temporales ni secretos.
